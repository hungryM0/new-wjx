# new-wjx

`new-wjx` 是一组针对 `wjx.cn`（问卷星）页面的轻量级 userscript / 工具集合。它基于作者之前的仓库思想进行整理，提供更清晰的文档、安装步骤与开发说明。

**项目简介**
- 通过 userscript 可对常见表单流程进行自动填写或优化，节省重复操作时间并提高可靠性。本仓库汇总了脚本、工具与使用说明，便于维护与迭代。

**功能**
- 自动填充或预设常用字段。
- 在允许范围内跳过非必要提示。
- 提供数据提取与提交的工具函数。

**安装**

- Userscript（推荐）：
  1. 安装 userscript 管理器，例如 Tampermonkey（Chrome/Edge）或 Violentmonkey/Greasemonkey（Firefox）。
  2. 从本仓库安装 userscript 文件（例如 `new-wjx.user.js`），可直接在浏览器中打开文件或使用管理器的“通过 URL 安装”功能。

- 独立工具/脚本：如果仓库包含辅助脚本，请按对应文件的使用说明在本地运行。注意检查并了解 `push.bat` 或其他自动化脚本的行为后再运行。

**使用方法**

- 安装 userscript 后，打开 `wjx.cn` 的问卷页面，脚本会自动运行并显示简洁的 UI 或执行配置的动作。
- 对于手动工具，请按照各脚本的说明运行。

**配置**

- 在 userscript 顶部修改常量以设置默认值（例如自动答案或字段值）。
- 切勿将敏感或个人信息提交到仓库，使用本地配置或环境变量保存密钥/秘密数据。

**开发**

- 需求：现代浏览器 + userscript 管理器用于运行测试。编辑使用任意代码编辑器即可。
- 推荐开发流程：
  1. 修改 userscript 文件。
  2. 在 userscript 管理器中加载脚本（开发期间可以使用自动更新 URL）。
  3. 在 `wjx.cn` 页面上测试并迭代。
